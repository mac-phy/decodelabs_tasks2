Key Concepts

Encryption formula: E(x) = (ord(char) − base + shift) % 26 + base
Decryption formula: D(x) = (ord(char) − base − shift) % 26 + base
Why % 26? The alphabet wraps around — shifting Z by 3 gives C, not a character beyond Z.
Why two bases? Uppercase A = ASCII 65, lowercase a = ASCII 97. We subtract the base to work in 0–25, apply the shift, then add the base back.
Weakness: Only 25 possible keys — an attacker can brute-force all shifts in seconds. Caesar cipher demonstrates the concept of symmetric encryption but is not used in real security systems.
